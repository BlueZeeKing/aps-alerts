# APS Alerts

An alert system designed for the [APS](https://apsva.us) school system alerts. The project is designed to be run frequently and will send a Discord alert when an alert it detected.

## Setup

### Configuration

`.env` file:

```properties
PROD_WEBHOOK=""
DEV_WEBHOOK=""
ERROR_WEBHOOK=""
PROD_ALERT_URL=""
DEV_ALERT_URL=""
```
